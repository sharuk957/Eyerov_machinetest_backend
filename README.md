# Eyerov_machinetest_backend
=
# Prerequisites
- python
- pip
- Django
# installation
```bash
git clone "https://github.com/sharuk957/Eyerov_machinetest_backend"
cd Eyerov_machinetest_backend
pip install django
pip install djangorestframework
python manage.py runserver
```
